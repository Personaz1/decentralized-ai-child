# How to Contribute

1. **Fork** the repository.
2. **Branch** naming: `feature/your-feature` or `fix/your-fix`.
3. **Test** changes with `pytest tests/`.
4. **Submit PR** with a clear description.

## Label System
- `good first issue`: Beginner-friendly
- `priority/P0`: Critical path tasks
- `needs-design`: Requires RFC discussion

## Code Standards
- **Python**: Black-formatted, type-hinted
- **Security**: All crypto must be audited
